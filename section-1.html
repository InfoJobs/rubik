Blockquotes

Use tag <blockquote> intead of <p> to introduce a block of text that is quoted from another source.
Use tag <cite> to cite the reference.

Use class .quote to give format to a blockquote.
Use class .vip-quote to emphasise a blockquote.

Lists

Use the following tags to create semantic lists:

  • <ul> for unordered lists
  • <ol> for ordered lists
  • <dl> for definition lists


Use class .list-default for a basic list without decoration.

  • Experiencia minima: no requerida
  • Tipo de oferta: de duracion determinada, jornada completa
