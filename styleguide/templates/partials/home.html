Rubik frontend framework

Rubik es el framework interno del portal InfoJobs. Rubik permite utilizar componentes CSS y elementos Javascript para construir las páginas del portal.

Objetivos de Rubik:

  1. - Estandarizar y reutilizar elementos y componentes.
  2. - Hacer más autónomo al equipo de desarrollo.

¿En qué se basa Rubik?

Rubik se basa en cuatro pilares básicos.

Mobile first

Mobile first se centra en jerarquizar y estructurar el contenido partiendo de la resolución de una pantalla móvil.

Responsive design

Responsive design ofrece una correcta visualización y experiencia en las distintas resoluciones y dispositivos.

OOCSS

Oriented Object CSS promueve la creación de componentes reutilizables, escalables y fáciles de mantener, separando los estilos que definen la estructura de los que definen la presentación.

Sass

Syntactically Awesome Stylesheets es un preprocesador CSS que simplifica la escritura y estructura de las hojas de estilo dando como resultado una CSS más limpia y ordenada.

Javascript

Javascript nos ayuda a enriquecer la interfaz e implementar comportamientos a nivel de interacción.

HTML semántico

El HTML semántico se utiliza para reforzar la coherencia entre las etiquetas HTML y el contenido, haciendo un uso correcto de los estándares web.
